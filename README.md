# SaveDouban

下载豆瓣加心歌曲的简单程序，需要.NetFramework3.5。


下载的音乐可以内嵌歌曲各项信息，包括歌手，专辑，发行年，豆瓣评分，甚至专辑封面。可以同时下载专辑封面和歌词。需查询并输入要下载的歌曲数目。若有下载失败的歌曲，请在相同的文件夹过几个小时到一两天再下载，会自动跳过下载成功的歌曲，不再重复下载。
